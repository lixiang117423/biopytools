# 重复序列屏蔽工具|Repeat Masking Tool

## 功能描述|Description

该模块用于基因组重复序列的识别和屏蔽，整合了 RepeatModeler 和 RepeatMasker 两个工具。

This module is for genome repeat identification and masking, integrating RepeatModeler and RepeatMasker tools.

## 工作流程|Workflow

1. **RepeatModeler** - 从头识别重复序列，构建物种特异性重复序列库
2. **RepeatMasker** - 使用重复序列库屏蔽基因组中的重复元件

## 安装依赖|Dependencies

- RepeatModeler v2.0.7
- RepeatMasker v4.1.2
- BuildDatabase (包含在 RepeatModeler 中)

## 使用方法|Usage

### 基本用法|Basic Usage

```bash
# 完整流程：RepeatModeler + RepeatMasker
biopytools repeatmask -i genome.fa -o output

# 指定物种(使用Dfam/Repbase数据库)
biopytools repeatmask -i genome.fa -o output -s arabidopsis

# 跳过RepeatModeler，仅运行RepeatMasker
biopytools repeatmask -i genome.fa -o output --skip-modeler

# 使用更多线程加速处理
biopytools repeatmask -i genome.fa -o output -t 24
```

### 参数说明|Parameters

| 参数|说明|默认值|
|------|------|--------|
| `-i, --genome` | 基因组FASTA文件(必需)|Genome FASTA file (required)|- |
| `-o, --output-dir` | 输出目录|Output directory|./repeatmask_output|
| `-t, --threads` | 线程数|Number of threads|12|
| `-s, --species` | 物种名称|Species name for Dfam/Repbase|None|
| `--skip-modeler` | 跳过RepeatModeler步骤|Skip RepeatModeler|False|
| `--no-dfam` | 不使用Dfam数据库|Do not use Dfam database|False|
| `--masking-mode` | 屏蔽模式: soft(小写)/hard(N)/x(X)|Masking mode|soft|
| `--use-ltr` | 运行LTR结构发现|Run LTR structural discovery|False|
| `--modeler-quick` | RepeatModeler快速模式|RepeatModeler quick mode|False|

### 屏蔽模式说明|Masking Modes

- **soft** - 使用小写字母表示重复序列(推荐用于基因预测)|Lowercase for repeats (recommended for gene prediction)
- **hard** - 使用N代替重复序列|Use N for repeats
- **x** - 使用X代替重复序列|Use X for repeats

## 输出文件|Output Files

### RepeatModeler输出
- `{db_name}-families.fa` - 重复序列一致性序列库|Repeat consensus library
- `builddatabase.log` - 数据库构建日志|Database building log
- `repeatmodeler.log` - RepeatModeler运行日志|RepeatModeler running log

### RepeatMasker输出
- `{genome}.masked` - 屏蔽后的基因组|Masked genome
- `{genome}.out` - 重复序列注释文件|Repeat annotation file
- `{genome}.out.gff` - GFF格式注释|GFF format annotation
- `{genome}.tbl` - 重复序列统计表|Repeat statistics table
- `repeatmasker_run.log` - 运行日志|Running log

## 示例|Examples

### 植物基因组(如水稻)|Plant Genome (e.g., Rice)

```bash
biopytools repeatmask \
  -i rice_genome.fa \
  -o rice_repeat_output \
  -s rice \
  -t 24 \
  --masking-mode soft
```

### 动物基因组(如果蝇)|Animal Genome (e.g., Fruit Fly)

```bash
biopytools repeatmask \
  -i drosophila.fa \
  -o drosophila_output \
  -s drosophila \
  -t 16 \
  --use-ltr
```

### 仅使用RepeatMasker(已有重复序列库)|RepeatMasker Only (have repeat library)

```bash
# 先用RepeatModeler生成库
biopytools repeatmask -i genome.fa -o step1_output

# 后续分析只需使用RepeatMasker
biopytools repeatmask -i genome.fa -o step2_output --skip-modeler -s species_name
```

## 注意事项|Notes

1. **物种名称**: 使用标准的拉丁学名或英文名，如 `arabidopsis`, `rice`, `maize`, `human`, `mouse` 等
2. **运行时间**: RepeatModeler 可能需要数小时到数天，取决于基因组大小
3. **内存需求**: 建议至少 32GB RAM，大基因组可能需要更多
4. **LTR分析**: `--use-ltr` 选项会显著增加运行时间，但能更全面识别LTR元件

该模块实现了相同的工作流程:

This module implements the same workflow described in the paper:
1. RepeatModeler generates species-specific repeats
2. RepeatMasker combines Dfam/Repbase database with RepeatModeler library for masking
